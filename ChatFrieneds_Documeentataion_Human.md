## Что такое Chat Friends?

**Chat Friends** — это корпоративный мессенджер для закрытого круга пользователей. Приложение позволяет общаться в личных и групповых чатах, обмениваться текстовыми сообщениями, изображениями и файлами.

### Основные возможности

✅ **Регистрация и вход** — быстрая регистрация по номеру телефона с возможностью загрузки аватара  
✅ **Личные чаты** — общение один на один с любым пользователем  
✅ **Групповые чаты** — создание групповых бесед с несколькими участниками  
✅ **Отправка файлов** — изображения, документы (PDF, DOC, DOCX и др.), а также видео и аудио файлы  
✅ **Предпросмотр медиа** — мини-превью изображений, видео и аудио перед отправкой  
✅ **Просмотр медиа** — полноэкранный просмотр изображений с возможностью сохранения в галерею, просмотр видео и прослушивание аудио прямо в чате  
✅ **Скачивание файлов** — сохранение документов на устройство  
✅ **Уведомления о новых сообщениях** — звук, вибрация и системное уведомление; при свёрнутом или закрытом приложении — фоновая проверка с уведомлением «Чат Друзей»; по нажатию на уведомление открывается список чатов  
✅ **Автоматические обновления** — проверка и установка новых версий приложения

## Начало работы

### Первый запуск

1. **Откройте приложение** — при первом запуске вы увидите экран входа
2. **Зарегистрируйтесь** — нажмите кнопку "Регистрация"
3. **Заполните данные:**
   - Номер телефона (обязательно)
   - Пароль (минимум 3 символа)
   - Имя (обязательно)
   - Фамилия, отчество, никнейм (по желанию)
   - Аватар (можно выбрать фото из галереи)
4. **Войдите** — после регистрации вы автоматически войдёте в систему

### Вход в систему

Если вы уже зарегистрированы:

1. Введите номер телефона
2. Введите пароль
3. Нажмите "Войти"

Приложение запомнит вас — при следующем запуске вход произойдёт автоматически.

## Основные функции

### 📋 Список чатов

После входа вы увидите список всех ваших чатов, разделённый на две вкладки:

- **Личные** — чаты один на один с другими пользователями
- **Групповые** — групповые беседы

**Особенности:**
- Чаты с непрочитанными сообщениями отображаются первыми
- Зелёный индикатор показывает наличие непрочитанных сообщений
- Превью последнего сообщения отображается под названием чата
- Потяните список вниз для обновления

**Навигация:**
- **Тап по чату** — открыть переписку
- **Кнопка "+"** (внизу справа) — создать новый чат
- **Иконка профиля** (вверху справа) — открыть свой профиль

### 💬 Экран чата

**Отправка сообщений:**
- Введите текст в поле ввода внизу экрана
- Нажмите кнопку отправки или Enter

**Отправка медиа и файлов:**
1. В нижней части экрана есть кнопка **вложений** слева от поля ввода сообщения.  
2. После нажатия откроется меню с вариантами:
   - Фото из галереи  
   - Фото с камеры  
   - Видео из галереи  
   - Аудио файл (MP3, M4A и др.)  
   - Документ или другой файл (PDF, DOC, и т.д.)
3. После выбора вы увидите **предпросмотр** выбранного файла над полем ввода:
   - Для фото — миниатюра изображения  
   - Для видео — плитка с иконкой воспроизведения  
   - Для аудио — небольшой плеер с кнопкой Play/Pause и полосой прогресса  
4. При отправке файл загружается на сервер и появляется в чате:
   - Фото — как картинка, по нажатию раскрывается на весь экран  
   - Видео — карточка с иконкой Play, по нажатию открывается экран воспроизведения  
   - Аудио — встроенный плеер прямо в сообщении  
   - Документы — карточка файла с названием и размером, по нажатию файл скачивается/открывается

**Работа с медиа:**
- **Тап по изображению** — открыть на весь экран с возможностью зумирования  
- **Тап по видео** — открыть видеоплеер и просмотреть ролик  
- **Тап по аудио-плееру** — запустить или остановить воспроизведение  
- **Тап по файлу** — скачать и открыть документ

**Обновление сообщений:**
- Сообщения обновляются автоматически каждые 5 секунд
- При возврате в список чатов данные обновляются

**Уведомления о новых сообщениях:**
- Когда вам приходит новое сообщение (и вы не находитесь в этом чате), приложение может показать **уведомление** со звуком и вибрацией
- Если приложение **свёрнуто или закрыто**, проверка новых сообщений выполняется примерно раз в 15 минут; при появлении новых сообщений вы получите уведомление «Чат Друзей» со звуком и вибрацией
- **Нажатие на уведомление** открывает экран со списком всех чатов (личные и групповые)
- При первом использовании разрешите приложению показывать уведомления (запрос появится автоматически)

### 👥 Создание чата

1. Нажмите кнопку "+" в правом нижнем углу списка чатов
2. Выберите тип чата:
   - **Личный чат** — общение с одним пользователем
   - **Групповой чат** — беседа с несколькими участниками
3. Выберите участников из списка
4. Для группового чата введите название
5. Нажмите "Создать"

После создания чата вы автоматически перейдёте в переписку.

### 👤 Профиль

В профиле отображается:
- Ваш аватар
- ФИО
- Никнейм (если указан)
- Номер телефона
- Дата регистрации

**Выход из аккаунта:**
- Нажмите кнопку "Выйти" внизу экрана профиля

## Руководство пользователя

### Как отправить изображение?

1. Откройте чат
2. Нажмите иконку изображения (📷) слева от поля ввода
3. Выберите "Галерея" или "Камера"
4. Выберите или сделайте фото
5. Изображение автоматически отправится

### Как отправить документ?

1. Откройте чат
2. Нажмите иконку файла (📎) слева от поля ввода
3. Выберите файл из списка (PDF, DOC, DOCX и др.)
4. Файл автоматически загрузится и отправится

### Как сохранить изображение в галерею?

1. Откройте чат с изображением
2. Нажмите на изображение для просмотра на весь экран
3. Нажмите кнопку сохранения (внизу экрана)
4. Разрешите доступ к галерее при первом использовании

### Как скачать файл?

1. Откройте чат с файлом
2. Нажмите на файл
3. Файл автоматически скачается в папку "Download"
4. После скачивания файл откроется автоматически

### Как проверить обновления приложения?

1. Откройте экран входа
2. Нажмите кнопку "Проверить обновление"
3. Если доступна новая версия, она автоматически скачается
4. После скачивания нажмите на уведомление для установки

### Как работают уведомления о новых сообщениях?

- **Приложение открыто:** новые сообщения проверяются каждые 5 секунд; при новом сообщении в чате, в котором вы не находитесь, показывается уведомление со звуком и вибрацией.
- **Приложение свёрнуто или закрыто:** проверка выполняется примерно раз в 15 минут (зависит от системы); при новых сообщениях приходит уведомление «Чат Друзей» с текстом «Новое сообщение», со звуком и вибрацией.
- **Нажатие на уведомление** всегда открывает общий список чатов (вкладки «Личные» и «Групповые»).
- Убедитесь, что в настройках устройства для приложения «Чат Друзей» разрешены уведомления и звук.

### Работа с API

Все запросы к серверу выполняются через класс `ApiService` (`lib/services/api_service.dart`).

**Пример использования:**

```dart
// Вход в систему
final response = await ApiService.login(phone, password);
if (response['success'] == true) {
  // Успешный вход, токен сохранён автоматически
}

// Получение списка чатов
final chats = await ApiService.getChats();

// Отправка текстового сообщения
final message = await ApiService.sendTextMessage(chatId, 'Привет!');

// Отправка файла
final file = File('/path/to/file.pdf');
final uploadResult = await ApiService.uploadFile(file);
final message = await ApiService.sendMessageWithFile(
  chatId, 
  'Смотри документ', 
  file, 
  'file'
);
```

## Технические детали

### Архитектура

Приложение использует стандартную архитектуру Flutter:
- **StatefulWidget** для экранов с состоянием
- **StatelessWidget** для простых виджетов
- **Сервисы** для бизнес-логики и работы с API
- **Модели** для представления данных

### Хранение данных

- **Токен авторизации** — SharedPreferences (ключ `token`)
- **Медиа-метаданные** — SharedPreferences (MediaStorage)
- **Локальные непрочитанные** — SharedPreferences (LocalUnreadHelper)
- **Состояние уведомлений** — SharedPreferences (lastKnownMessageIds, currentUserId для фоновой проверки)
- **Файлы** — папка Download на устройстве

### Обновление данных

- **Список чатов** — обновляется при открытии экрана и при возврате из чата
- **Сообщения в чате** — автоматическое обновление каждые 5 секунд (polling)
- **Pull-to-refresh** — ручное обновление списка чатов

### Обработка ошибок

- Все ошибки API логируются в консоль
- Пользователю показываются понятные сообщения об ошибках
- При сетевых ошибках приложение продолжает работать с локальными данными

### Безопасность

- Токены хранятся локально в SharedPreferences
- Все запросы к API используют HTTPS
- Пароли передаются в открытом виде (планируется хеширование на сервере)

## Часто задаваемые вопросы

### Как восстановить пароль?

В текущей версии восстановление пароля не реализовано. Обратитесь к администратору системы.

### Почему не приходят сообщения сразу?

В открытом чате сообщения обновляются каждые 5 секунд. Уведомления о новых сообщениях приходят при опросе (каждые 5 сек, если приложение открыто) или при фоновой проверке (примерно раз в 15 минут, если приложение свёрнуто или закрыто). В будущих версиях возможен переход на WebSocket для мгновенной доставки.

### Можно ли использовать приложение без интернета?

Нет, приложение требует постоянного подключения к интернету для работы с сервером.

### Где хранятся скачанные файлы?

Все файлы сохраняются в папку **Download** на вашем устройстве. Вы можете найти их через файловый менеджер.

### Как изменить аватар после регистрации?

В текущей версии изменение аватара после регистрации не реализовано. Эта функция планируется в будущих обновлениях.
